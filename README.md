# Slopblock for Hacker News

Community contributed blocklist for hiding comments from accounts that primarily contribute LLM-driven comments on [Hacker News](https://news.ycombinator.com).

## Installing

Begin by getting a userscript manager for your browser. You may use [Tampermonkey](https://www.tampermonkey.net/) for Chrome or Firefox, or [Userscripts](https://github.com/quoid/userscripts) for Safari.

Then, install the userscript from https://github.com/supriyo-biswas/hn_slopblock/raw/refs/heads/master/hn_slopblock.user.js

## Contributing

Please raise a pull request with the username in question and link comments as evidence. Similarly, if you think a particular entry is in error, you may raise a pull request, providing evidence to support your assertion.

**We reserve the right to decline your contribution with or without providing a response.**
